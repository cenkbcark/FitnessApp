# FitnessApp

```html
It's a Fitness App which provides you best exercises according to data.
```


<h2>Architecture</h2>

```html
I preferred clean MVC pattern.
```


<h2>Features</h2>

```html
Onboarding screen.
CollectionView and Flow Layout.
PickerView for selecting height and weight.
Lottie for Animation.
AVKit - AVFoundation
```
